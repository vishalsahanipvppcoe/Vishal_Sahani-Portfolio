# Vishal Sahani Portfolio

A modern and responsive developer portfolio built using Next.js, TypeScript, Tailwind CSS, MongoDB, and Nodemailer.

## 🚀 Features

* Responsive modern UI
* Animated sections
* Contact form integration
* MongoDB database connection
* Email sending with Nodemailer
* Dark/Light mode
* Fast performance with Next.js

## 🛠️ Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* MongoDB
* Nodemailer
* React Icons

## 📂 Project Structure

```bash
app/
components/
config/
constants/
hooks/
lib/
model/
public/
types/
```

## ⚙️ Environment Variables

Create a `.env.local` file in the root directory and add:

```env
NODEMAILER_HOST=
NODEMAILER_PORT=
NODEMAILER_USER=
NODEMAILER_PASS=
MONGODB_URI=
```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/vishalsahanipvppcoe/Vishal_Sahani-Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
